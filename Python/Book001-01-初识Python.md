* 访问

* 列表方法
  * append 在列表末尾增加一个数据项
  * pop 删除列表末尾的一个数据项
  * extend 在列表末尾增加一个数据项集合
  * remove 在列表中找到并删除一个数据项
  * insert 在某个特定的位置前面增加一个数据项
